# JP-EN Plant Translator Concept

## Purpose

This project concept demonstrates glossary-controlled Japanese-to-English translation for manufacturing environments.

## Core Workflow

```text
User input
-> Japanese text normalization
-> internal glossary matching
-> manufacturing context prompting
-> OpenAI translation
-> reviewable English output
```

## Example Terms

- Butterfly valve
- PLC heartbeat signal
- Master ON
- Battery cover

These are examples of manufacturing engineering terminology, not generic language terms. Their correct translation depends on equipment, controls, product, and plant context.

## Why It Matters

The goal is to shift AI translation from generic language conversion to governed manufacturing semantics.
