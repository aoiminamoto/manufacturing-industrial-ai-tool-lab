# PLC Fix AI Assistant Concept

## Purpose

A future PLC Fix AI Assistant could use governed plant knowledge to help maintenance members identify possible root causes and recommended checks from natural-language descriptions of machine conditions, NG situations, and operational symptoms.

This is especially valuable in multilingual manufacturing plants, where interpreting equipment issues consistently across languages is often difficult.

## Input

Maintenance members describe issues in natural language:

- Machine condition
- NG situation
- Symptoms
- Alarm / signal behavior

## Knowledge Used

The assistant references governed plant knowledge:

- PLC / HMI comments
- Standardized NG records
- Equipment and process context
- Previous troubleshooting history

## Output

The assistant provides:

- Possible root causes
- Recommended checks
- Related alarm / NG context
- Previous countermeasures

## Safety Note

The assistant supports maintenance decision-making but does not automatically control equipment or replace required engineering judgment.
